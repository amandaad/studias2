## Estrutura

- `studioas/index.html`
- `studioas/test.css`
- `studioas/assets/*`

## Como rodar localmente

1. Entre na pasta do projeto:
   - `cd studioas`
2. Inicie um servidor local:
   - `python -m http.server 5500`
3. Abra no navegador:
   - `http://localhost:5500`
